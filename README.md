# AutoClaw 自定义技能库

本仓库存放 AutoClaw 的自定义技能（Skills）。

## 技能列表

### 图片识别
- **baidu-ocr** — 百度 OCR 图片识别，支持中英文混合识别、股票持仓截图、表格等

### 语音处理
- **baidu-speech** — 百度语音合成与识别
- **whisper-speech** — OpenAI Whisper 语音识别
- **feishu-voice** — 飞书语音消息发送

### 通信工具
- **email-sender** — 邮件发送功能

### 文档处理
- **word-generator** — Word 文档生成

### 同步工具
- **github-sync** — GitHub 同步功能

### 开发工具
- **skill-creator** — AutoClaw 技能创建工具

---

由 AutoClaw 自动生成并同步
